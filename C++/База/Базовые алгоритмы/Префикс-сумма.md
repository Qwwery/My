Префикс сумма - это подход, когда заранее вычисляется вся сумма элементов массива
обычно префикс сумма - это дополнительный массив

Для использования префикс суммы надо уметь:
1) Уметь быстро создавать массив префикс суммы
2) Уметь быстро отвечать на запрос суммы элементов с позиции l до r
## Создание
т.к. у нас есть формула 
$$
b_i+_1 = b_i + a_i+_1
$$
Решение 
```cpp
int n;
cin >> n;
vector<int> A(n);
for (int i = 0; i < n; i++)
	cin >> A[i];
vector<int> Pref(n + 1, 0);
for (int i = 0; i < n; i++)
	Pref[i + 1] = A[i] + Pref[i];
```

## Ограничения
1) Все операции, для заполнения префикс суммы, должны быть обратимыми (для + это -, для * это /)
2) Массив A(исходный) не должен меняться в процессе. Изменение хотя бы одного элемента, то префикс функция будет полностью меняться.

Задача: Надо найти сумму элементов в прямолинейной фигуре на матрице
Решение: Создаем Матрицу-Префикс-Сумму в b_i_j ячейке будет храниться вся сумма ячеек выше и левее 

```cpp
int main() {
	int n, m;
	cin >> n >> m;
	vector<vector<int>> A(n + 1, vector<int>(m + 1));
	for (int i = 1; i <= n; i++)
		for (int j = 1; j <= m; j++)
			cin >> A[i][j];
	vector<vector<long long>> B(n + 1, vector<long long>(m + 1));
	for (int i = 1; i <= n; i++)
		for (int j = 1; j <= m; j++)
			B[i][j] = B[i][j - 1] + A[i][j];
	for (int i = 1; i <= n; i++)
		for (int j = 1; j <= m; j++)
			B[i][j] = B[i - 1][j];
	int q, xl, xr, yr, yl;
	long long res;
	cin >> q;
	for (int i = 0; i < q; i++) {
		cin >> xl >> xr >> yr >> yl;
		res = B[xr][yr] - B[xl - 1][yl] - B[xr][yl - 1] + B[xl - 1][yl - 1];
		cout << res << '\n';
	}
}
```

Для увеличения множества элементов из массива длинной n можно использовать следующий код
```cpp
int main() {
	int n;
	cin >> n;
	vector<int> A(n + 1);
	for (int i = 1; i <= n; i++)
		cin >> A[i];
	// здесь можно прописать через for несколько таких изменений
	cin >> l >> r >> c;
	Pref[l] += c;
	Pref[r + 1] -= c;
	for (int i = 1; i <= n; i++)
		A[i] = A[i - 1] + Pref[i];
}
```
