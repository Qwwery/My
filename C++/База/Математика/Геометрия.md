Одна из основных тематик в спорт проге в подразделе [[Математика||Математики]].
Основные проблемы работы с вещественными числами:
1) Вывод вещественного числа с нужной точностью
2) Потеря точности с операциями вещественных чисел
Для решения первой проблемы, необходимо понимать, как работает тестирующая система при проверке ответа участника и члена жюри.
Если в условии задачи указано выведите ответ с точностью 4 знака после запятой. Тестирующая система проверяет по формуле |res_jury - res| < 0.0001.
Банальное хорошее решение проблемы выводить ответ с заведомо больших количеством цифр после запятой.
```
#include <iomanip>
cout << fixed << setprecision(10)<<a;
```
Вторая проблема потеря точности связанна с представлением мантиссы у вещественных чисел. К сожалению, мантисса конечна при любой операции вещественных чисел существует ошибка в конце мантиссы. Если операции с вещественными числами большое количество ошибка накапливается и может стать существенной проблемой при сборе ответа.

Решение - стараться не использовать вещественные числа.

## Структуры
Шаблон структур C++
```
struct name_struct {
	компоненты
};
```
Доступ к членам структуры. Для доступа к членам структуры используется оператор (.)
Членами структуры считаются:
Поля - элементы, которые хранят информацию
Методы - внутренние функции структуры

## Точка 
Точка - это геометрическая фигура, из которой состоят все остальные геометрические фигуры.
## Вектор 
Вектор - это направленный отрезок, т.е. отрезок, имеющий длину и определенное название
Точка и вектор задаются в пространстве R^n при помощи координат.

```
struct point {
	double x, y;
	point(double curx = 0, double cury = 0) {
		x = curx;
		y = curx;
	}
};
```
Конструкторы представляют специальную функцию, которая имеет то же имя, что и класс, которая не возвращает никакого значения и которая позволяют инициализировать объект класса во время его создания и таким образом гарантировать, что поля класса будут иметь определенные значения.

Перегрузка операторов позволяет определить для объектов классов встроенные операторы, такие, как "+", "-", "\*" И т.д. Для определения оператора для объектов своего класса, необходимо определить функцию, название которой содержит слово operator и символ перегружаемого оператора
```
point operator+ (point TMP) {
	point res;
	res.x = x + TMP.x;
	res.y = y + TMP.y;
	return res;
}

point operator- (double k) {
	point res;
	res.x = x * k;
	res.y = y * k;
	return res;
}
```
Пример работы:
```
point A, B(3, 5);
point C = B;
A = B + C;
```

Полезное с векторами:
1) Нахождение длины вектора 
$$|a| = sqrt(a.x * a.x + a.y * a.y)$$
2) Нахождение скалярного произведения
$$
(a, b) = |a| * |b|cos(O) = a.x * b.x + a.y * b.y
$$
3) Нахождение векторного произведения
$$
|a, b| = |a| * |b|sin(O) = a.x * b.y - a.y * b.x
$$
Полезные функции:
```
double dist2(point A) 
	return A.x * A.x + A.y * A.y

double dot(point A, point B)
	return A.x * B.x + A.y * B.y;

double cross(point A, point B)
	return A.x * B.y - A.y * B.x;
```
Операция нормирования вектора - это изменение длины вектора до длины равной 1 (его уменьшение или сжатие на нужную величину)
Операция поворот вектора на угол O
$$
new_a = \frac{a}{|a|}
$$

## Треугольник
В задачах по спорт прогу сложные 
Площадь треугольника
1) Формула Герона:
$$
p = \frac{a + b + c} {2}
$$
$$
S = \sqrt{p * (p - a) * (p - b) * (p - c)}
$$
Не пойдет из-за переполнения типа данных
2) Формула сторона на высоту
$$
S = \frac{1}{2} * h * a
$$
К сожалению надо знать высоту
3) Формула две стороны и угол между ними 
$$
S = \frac{1}{2} * a * b * sin(O) = \frac{|cross(a, b)|}{2}
$$

Высота треугольника определяется через площадь
Медиана определяется через координату окончания медианы 
Биссектриса  сложно задается

## Прямая 
Общее уравнение прямой линии на плоскости в декартовых координатах:
$$
A_x + B_x + C = 0
$$
Так же это уравнение называют нормальным уравнением прямой, так как по коэффицентам A, B. Определяется вектор ~58
