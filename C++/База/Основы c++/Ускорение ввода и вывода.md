В задачах, где много вывода, можно получить TL только из-за того, что ввод (или вывод) занимает очень много времени. По умолчанию C++ использует синхронизацию с stdio, что может замедлять операции ввода-вывода. Можно отрубить её, используя специальную команду:
```
ios::sync_with_stdio(0);
cin.tie(0);
```
