Одна из [[Структуры данных||структур данных]] в языке c++, которая активно используется в спорт проге.
Deque - Двухсторонняя очередь, её называют двухглавой змеей.
В некотором смысле deque улучшенная версия vector. Мы можем также обращаться по элементам при помощи операторов []
Пример: котопёс
```
.empty()     - Проверка на пустоту
.size()      - Размер
.pop_back()  - Удалить с конца
.pop_front() - Удалить с начала 
.push_back() - Добавить в конец
.push_fron() - Добавить в начало 
.fron()      - Показать верхний
.back()      - Показать нижний

нижу указывают на:
.begin()     - Первый элемент 
.end()       - За последний элемент
.rbegin()    - На последний
.rend()      - Перед первым элементом
```
Данный метод не прижился.
Из-за операций индексация постоянно смещается 
Можно использовать sort()
