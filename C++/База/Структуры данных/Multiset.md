[[Структуры данных]] в языке C++

В мульти сете действует только 1 правило из set, это упорядоченность по возрастанию 
Одно свойство multiset
1) Все элементы упорядочены

```
multiset<int> st;
st.insert(10);
int tmp = *st.begin();
```
## Методы
[[Set#^89e701||Те же, что и у set]]
Пример отличия .erase(a) и .erase(iter)
```
multiset<int> st = {1, 1, 2, 2, 3, 4, 7, 9, 12}
```

```
st.erase(1); // 2, 2, 3, 4, 7, 9, 12
```

```
st.erase(st.begin()); // 1, 2, 2, 3, 4, 7, 9, 12
```
