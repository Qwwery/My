Одна из [[Структуры данных||структур данных]] в языке c++, которая активно используется в спорт проге.
Pair - это контейнер, который объединяет два элемента одного или разных типов данных. Он состоит из двух элементов: Первого и второго (порядок должен быть фиксированным)
```
pair<int, string> my_p;
my_p.first = 10;
my_p.second = "123";
```
Pair может быть вложенным
```
pair<pair<int, string>, pair<int, int>> monster;
monster.first.second = "123";
monster.first.first = 10;
monster.second.first = 15;
monster.second.second = 11;
```
Pair можно сортировать
Сортировка будет сначала по первому элементу, потом по второму
