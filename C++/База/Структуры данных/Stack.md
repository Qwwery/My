Одна из [[Структуры данных||структур данных]] в языке c++, которая активно используется в спорт проге.
Steck - структура данных, которая работает по принципу LIFO (last-in, first-out или последний вошел, первый вышел)
Пример: Стопка карт, книги, подносы в столовой.
Методы Steck'а:
```
.size()  - размер стопки
.empty() - проверка на пустоту
.push()  - добавить в конец
.top()   - обратиться на верхний элемент 
.pop()   - удалить верхний элемент
```
