# Скачивание g++

- Скачиваем MSYS2, подходящую под ОП, с официального [сайта](|https://www.msys2.org/).
- Открываем программу от имени администратора и вводим: 
```bash
pacman -S mingw-w64-x86_64-gcc
```
- Возможно потребуется подтверждение, на которое обязательно надо ответить `Y`.
- Ждем загрузку.

# Открытие консоли
1) Можно запустить командную строку, нажав комбинацию клавиш `win + R`, затем прописав cmd в появившемся окне.
2) Можно запустить используя поисковик windows. Прописав cmd и нажав на приложение
3) Можно нажать комбинацию клавиш `ctrl + shift + esc` в окне диспетчера задач слева сверху нажать на `файлы\Запустить новую задачу`
4) Можно открыть проводник любым способом перейти в `C:\Windows\System32`, после найти и запустить файл cmd

# Проверка установки
Для проверки установился ли g++ в консоли стоит прописать код:
```bash
g++ --version
```

Если консоль в ответ выведет что-то в духе этого:
```bash
g++ (Rev8, Built by MSYS2 project) 15.2.0
Copyright (C) 2025 Free Software Foundation, Inc.
This is free software; see the source for copying conditions.  There is NO
warranty; not even for MERCHANTABILITY or FITNESS FOR A PARTICULAR PURPOSE.

```
То тогда всё в порядке

# Запуск программы через консоль
- Переходим в директорию (папку), в которой лежит файл с кодом, который надо запустить
	примерное рассположение:
```bash
cd C:\Users\{имя пользователя}\OneDrive\Desktop\{папка с кодом}
```
- После перехода вводим:
```bash
g++ -O2 {код} -o {программа запуска}
```
Где `код` - файл, который запускаем. Пример: `in.cpp`
`программа запуска` - создаваемый файл, при открытии которого будет запускаться программа. Пример: `res.exe`
`-O2` - флаг для включения ускорения программы. Подробнее по [ссылке](https://gcc.gnu.org/onlinedocs/gcc/Optimize-Options.html)
