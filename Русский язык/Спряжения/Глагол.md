- **I спряжение** — окончания: **-у/-ю, -ем, -ешь, -ете, -ет, -ут/-ют**
- **II спряжение** — окончания: **-у/-ю, -им, -ишь, -ите, -ит, -ат/-ят**

# Определение спряжения
- Смотрим на букву в инфинитиве перед -ть
- Если там **А, О, Е, Ы** → **I спряжение**
- Если там **И** → **II спряжение**
##### Примеры:

- **брать** → А → I спряжение
- **носить** → И → II спряжение
- **видеть** → Е → I спряжение
- **стелить** → И → II спряжение

# Окончания глаголов по спряжению
| Лицо       | 1 спряжение | 2 спряжение |
| ---------- | ----------- | ----------- |
| я          | -у / -ю     | -у / -ю     |
| ты         | -ешь        | -ишь        |
| он/она/оно | -ет         | -ит         |
| мы         | -ем         | -им         |
| вы         | -ете        | -ите        |
| они        | -ут / -ют   | -ат / -ят   |

# Примеры применения

Пример 1: он нос**и**т
1. Инфинитив: носить
2. Перед -ТЬ: И
3. И = 2 спряжение
4. Он + второе спряжение = -ИТ
---
Пример 2: Я пиш**у**
1. Инфинитив: писать
2. Перед -ТЬ: А
3. А = 1 спряжение
4. Я + первое спряжение = -У или -Ю, Ю - не подходит, а потому У
---
Пример 3: Вы чита**ете**
1. Инфинитив: читать
2. Перед -ТЬ: А
3. А = 1 спряжение
4. Вы + первое спряжение = -ЕТЕ

# Исключения

#### Глаголы на -ЕТЬ, которые второго спряжения
1. Смотреть     -> он смотр**и**т
2. Видеть         -> он вид**и**т
3. Ненавидеть -> он ненавид**и**т
4. Обидеть       -> он обижа**е**т

Гнать, держать, смотреть и видеть,  
Дышать, слышать, ненавидеть,

#### Глаголы на -АТЬ / -ЯТЬ, которые второго спряжения
1. Зависеть -> он завис**и**т
2. Терпеть -> он терп**и**т
3. Вертеть -> он верт**и**т
4. Брезжить -> он брезж**и**т

брить, стелить.
